
Projecto desenvolvido em Laravel 10

Download PHP 8.1.13 (cli) https://www.php.net/downloads.php
    Instalar PHP     - Requer colocar manualmente a pasta (extraída) no disco C:
                     - Alterar as enviorment variables
                        -Edit system variables-> enviorment variables-> Path-> Edit-> New-> colocar caminho da pasta PHP (e.g. C:\php-8.1.13)-> OK
                     - Abrir cmd-> php -v (verificar que está corretamente instalado)

Download PHP Composer : https://getcomposer.org/Composer-Setup.exe
                     - verificar no cmd com composer -v
                     - cmd -> composer update

Download MySQL Installer : https://dev.mysql.com/downloads/installer/
                     - Instalar
                     - Download TablePlus (UI para MySQL)
                     

Cmd-> abrir pasta  do projecto (e.g C:\Users\Work\Desktop\roboyo_pentest_report_tool)
